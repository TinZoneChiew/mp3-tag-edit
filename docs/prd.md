# 需求文档

## 1. 应用概述

### 1.1 应用名称
MP3元信息编辑器

### 1.2 应用描述
一个在线网页工具,用户可上传MP3文件并编辑其元信息(如标题、艺术家、专辑、歌词等)及专辑封面,编辑完成后可下载修改后的文件。

## 2. 页面结构与功能说明

### 2.1 页面结构
```
├── 编辑页面
├── 封面预览弹窗
├── 歌词编辑弹窗
```

### 2.2 编辑页面

#### 2.2.1 文件上传区域
- 支持用户点击上传按钮选择本地MP3文件上传
- 支持用户直接拖拽MP3文件至上传区域完成上传
- 显示已上传文件的文件名

#### 2.2.2 元信息编辑区域
- 标题:文本输入框,可编辑歌曲标题
- 艺术家:文本输入框,可编辑艺术家名称
- 专辑:文本输入框,可编辑专辑名称
- 年份:文本输入框,可编辑发行年份
- 流派:下拉选择框,可从预设流派列表中选择音乐流派
- 备注:文本输入框,可编辑备注信息
- 歌词:显示歌词摘要或状态(如「已包含歌词」或「无歌词」),点击可打开歌词编辑弹窗

#### 2.2.3 专辑封面编辑区域
- 封面显示区域:显示当前MP3文件的专辑封面图片,若无封面则显示默认占位图
- 查看封面:点击封面图片打开封面预览弹窗
- 更换封面:提供上传按钮,用户可选择本地图片文件替换封面
- 移除封面:提供移除按钮,点击后删除当前封面

#### 2.2.4 操作区域
- 保存按钮:保存编辑后的元信息及封面
- 下载按钮:下载修改后的MP3文件

### 2.3 封面预览弹窗

#### 2.3.1 预览显示区域
- 以放大模式显示封面图片
- 图片居中显示
- 若MP3文件包含多张图片(如背面、光盘面、插图等),显示当前查看的图片

#### 2.3.2 图片切换功能
- 左箭头按钮:点击切换至上一张图片
- 右箭头按钮:点击切换至下一张图片
- 图片索引指示器:显示当前图片序号及总图片数量(如 1/3)
- 当仅有一张图片时,不显示切换按钮及索引指示器

#### 2.3.3 图片调整功能
- 旋转按钮:点击可将当前图片顺时针旋转90度
- 支持连续旋转,每次旋转90度

#### 2.3.4 操作按钮
- 关闭按钮:关闭预览弹窗返回编辑页面

### 2.4 歌词编辑弹窗

#### 2.4.1 歌词显示与编辑区域
- 多行文本输入框,显示当前MP3文件的歌词内容
- 若MP3文件不包含歌词,则显示为空白状态
- 用户可直接在文本框中编辑歌词内容

#### 2.4.2 操作按钮
- 保存按钮:保存编辑后的歌词内容并关闭弹窗
- 取消按钮:放弃修改并关闭弹窗
- 清空按钮:清空当前歌词内容

## 3. 业务规则与逻辑

### 3.1 文件处理流程
1. 用户通过点击上传按钮或拖拽方式上传MP3文件
2. 系统读取文件的元信息、专辑封面及歌词并显示在编辑区域
3. 用户修改元信息内容、编辑专辑封面或编辑歌词
4. 用户点击保存按钮,系统将修改后的元信息、封面及歌词写入文件
5. 用户点击下载按钮,下载修改后的MP3文件

### 3.2 文件上传规则
- 支持点击上传按钮选择文件
- 支持拖拽文件至上传区域
- 拖拽时上传区域显示视觉反馈(如高亮边框或背景色变化)
- 拖拽松开后自动开始上传并读取文件信息

### 3.3 元信息读写规则
- 上传文件后自动读取并显示现有元信息
- 系统读取标题、艺术家、专辑、年份、流派、备注、歌词等所有元信息字段
- 若某项元信息为空,则对应输入框或下拉框显示为空白状态
- 所有元信息字段均在编辑区域显示,无论原文件中是否包含该信息
- 保存时将用户输入的内容写入MP3文件的元信息标签

### 3.4 流派选择规则
- 流派下拉框提供常见音乐流派选项,包括但不限于:流行、摇滚、古典、爵士、电子、民谣、嘻哈、乡村、金属、蓝调等
- 若原MP3文件的流派信息在预设列表中,则自动选中对应选项
- 若原MP3文件的流派信息不在预设列表中或为空,则下拉框显示为空白状态

### 3.5 专辑封面处理规则
- 上传MP3文件后自动读取并显示专辑封面,若无封面则显示默认占位图
- 用户更换封面时,系统自动压缩上传的图片
- 用户移除封面后,封面显示区域恢复为默认占位图
- 保存时将封面数据写入MP3文件

### 3.6 多图片读取与显示规则
- 系统读取MP3文件中的所有图片(包括封面、背面、光盘面、插图等)
- 编辑页面的封面显示区域默认显示第一张图片
- 若文件包含多张图片,用户可通过预览弹窗查看所有图片

### 3.7 封面预览与调整规则
- 点击编辑页面的封面图片打开预览弹窗
- 预览弹窗中图片以放大模式显示
- 若MP3文件包含多张图片,用户可通过左右箭头按钮切换查看
- 切换至第一张图片时,左箭头按钮禁用或隐藏
- 切换至最后一张图片时,右箭头按钮禁用或隐藏
- 用户可通过旋转按钮将当前查看的图片顺时针旋转90度
- 旋转操作仅影响预览显示,不改变原始封面数据
- 关闭预览弹窗后返回编辑页面,旋转状态不保留

### 3.8 歌词读取与编辑规则
- 上传MP3文件后自动读取歌词信息
- 若文件包含歌词,编辑页面显示「已包含歌词」状态
- 若文件不包含歌词,编辑页面显示「无歌词」状态
- 点击歌词区域打开歌词编辑弹窗
- 歌词编辑弹窗中显示完整歌词内容,用户可自由编辑
- 点击保存按钮后,歌词内容写入MP3文件并关闭弹窗
- 点击取消按钮后,放弃修改并关闭弹窗
- 点击清空按钮后,清空当前歌词内容

## 4. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 用户上传非MP3格式文件 | 提示用户仅支持MP3格式文件 |
| 用户拖拽非MP3格式文件至上传区域 | 提示用户仅支持MP3格式文件 |
| 用户未上传文件直接点击保存或下载 | 提示用户请先上传MP3文件 |
| 元信息输入框为空 | 允许保存,对应元信息项保持为空 |
| 流派下拉框未选择 | 允许保存,流派信息保持为空 |
| 用户上传非图片格式文件作为封面 | 提示用户仅支持图片格式文件 |
| 用户移除封面后未上传新封面 | 允许保存,MP3文件不包含封面信息 |
| 点击默认占位图 | 不打开预览弹窗,仅当存在实际封面图片时可预览 |
| MP3文件仅包含一张图片 | 预览弹窗中不显示切换按钮及索引指示器 |
| MP3文件不包含任何图片 | 编辑页面显示默认占位图,点击不打开预览弹窗 |
| 歌词编辑弹窗中歌词内容为空 | 允许保存,MP3文件不包含歌词信息 |
| 用户点击清空按钮 | 清空歌词内容,用户可继续编辑或保存 |
| 用户点击取消按钮 | 放弃所有修改,关闭弹窗,歌词保持原状态 |
